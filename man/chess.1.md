# NAME

**chess** - Simple SDL2 chess program

# SYNOPSIS

**chess** [OPTION]...

# DESCRIPTION

**chess** is a simple chess program written in C using SDL2 and Nuklear. It supports playing vs a friend or an AI.

# OPTIONS

-h, \--help
: Shows help text

-V, \--version
: Shows the version
